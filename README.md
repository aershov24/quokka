# RecomenDo

RecomenDo. Suggest your best.
RecomenDo helps you to create, manage and share your best recommendations with the world.

## Tech Stack
Back-end:  
* NodeJS 
* Express 
* MongoDB 
* Mongoose

Front-end:  
* AngularJS
* Bootstrap 
* Jade
* Chay 
* Instanbul

## Installation
```
npm install
npm run nodemon
```
